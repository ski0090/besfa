# Agent Instructions

- Do not open pull requests unless the user explicitly asks for one.
- Read and follow `DOCUMENT.md` before changing public API, project structure,
  IPC protocol, runtime launch behavior, or documentation.
- When adding or changing APIs exposed with Rust `pub` or Dart public classes,
  keep public API documentation comments in the same change.
- When project behavior or ownership changes, update the relevant project
  `README.md` in the same change.
